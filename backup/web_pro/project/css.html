CSS (Cascading Style Sheets) คือ ภาษาที่ใช้สำหรับตกแต่งเอกสารแบบ HTML ใช้สำหรับตกแต่งเอกสารให้มีหน้าตา สีสัน ระยะห่าง พื้นหลัง เส้นขอบและอื่นๆ ตามที่คุณต้องการ ภาษา CSS มีลักษณะหรือรูปแบบในการเขียน Syntax ที่เฉพาะ และได้ถูกกำหนดมาตรฐานโดย W3C เป็นภาษาหนึ่งในการตกแต่งเว็บไซต์ที่ได้รับความนิยมอย่างแพร่หลาย ช่วยให้นักพัฒนาเว็บสามารถแยกการนำเสนอเว็บไซต์ออกจากโครงสร้าง ทำให้ควบคุมเลย์เอาต์ สี แบบอักษร และลักษณะการมองเห็นอื่นๆ ของหน้าเว็บได้ง่ายขึ้น ด้วย CSS คุณสามารถสร้างการออกแบบที่สอดคล้องกันและดึงดูดสายตาบนหน้าเว็บหลายหน้าได้

โดยมีรูปแบบที่นิยมใช้ทั้งหมด 3 แบบ และมีองค์ประกอบ ดังต่อไปนี้

  • Inline CSS คือ การเขียน CSS ไว้ภายในเอกสาร HTML โดยตรง รูปแบบนี้เหมาะสำหรับการกำหนดรูปแบบให้กับองค์ประกอบเพียงเล็กน้อย 
    h1 style="color:blue;text-align:center;">This is a heading/h1>
p style="color:red;">This is a paragraph./p>
  • Internal CSS คือ การเขียน CSS ไว้ในเอกสาร HTML เดียวกัน แต่แยกส่วนออกจากเนื้อหา HTML ในรูปแบบของ <style> 
    style>
body {
  background-color: linen;
}

h1 {
  color: maroon;
  margin-left: 40px;
}
/style>
  • External CSS คือ การเขียน CSS ไว้ในไฟล์แยกต่างหาก แล้วเชื่อมโยงกับเอกสาร HTML โดยใช้แท็ก Link 
    link rel="stylesheet" href="mystyle.css">
h1>This is a heading/h1>
p>This is a paragraph./p>

    
//mystyle.css
body {
  background-color: lightblue;
}

h1 {
  color: navy;
  margin-left: 20px;
}
