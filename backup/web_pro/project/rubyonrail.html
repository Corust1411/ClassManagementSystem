Ruby On Rails

Ruby on Rails (หรือเรียกสั้น ๆ ว่า Rails) เป็นเฟรมเวิร์ก (framework) สำหรับการพัฒนาเว็บแอปพลิเคชัน (web applications) ซึ่งใช้ภาษาโปรแกรม Ruby ในการเขียนโค้ดของมัน รูบี้ (Ruby) เป็นภาษาโปรแกรมที่โด่งดังและได้รับความนิยมจากความอ่านง่ายและการเขียนโค้ดที่เป็นมิตรกับผู้พัฒนา. Rails ถูกสร้างขึ้นโดย David Heinemeier Hansson ในปี 2003 และได้รับความนิยมมากในชุดเวลาต่อมา. เฟรมเวิร์กนี้เน้นการพัฒนาแบบ "Convention over Configuration" ซึ่งหมายความว่ามีค่าเริ่มต้นที่ถูกกำหนดไว้ล่วงหน้าเพื่อให้นักพัฒนาไม่ต้องกำหนดค่ามากมาย และต้องการให้โปรแกรมเมอร์มีโครงสร้างโค้ดที่เป็นระเบียบและมีความทันสมัย 

 Example Code Ruby On Rails# app/controllers/products_controller.rb

class ProductsController < ApplicationController
  def index
    @products = Product.all
  end

  def show
    @product = Product.find(params[:id])
  end

  def new
    @product = Product.new
  end

  def create
    @product = Product.new(product_params)

    if @product.save
      redirect_to @product, notice: 'Product was successfully created.'
    else
      render :new
    end
  end

  private

  def product_params
    params.require(:product).permit(:name, :description, :price)
  end
end

Ruby on Rails (RoR) มีหลายข้อดีที่ทำให้มันเป็นเฟรมเวิร์กที่นักพัฒนามักทำนิยม ดังนี้:


  • การพัฒนาที่รวดเร็ว: Rails เน้นการพัฒนาที่รวดเร็วและมีความสะดวกสบาย ด้วยความสามารถในการสร้างโค้ดได้เร็วทำให้นักพัฒนาสามารถพัฒนาและทดสอบแอปพลิเคชันได้โดยเร็ว.

  • Security Features: Rails มีความสนใจถึงด้านความปลอดภัย มีสิ่งที่เรียกว่า "ความปลอดภัยตามต้นทาง" (Security by Design) ทำให้มีการป้องกันตั้งแต่ขั้นตอนการพัฒนา.

  • Gem System: Rails มีระบบ Gem ที่ช่วยในการเพิ่มฟีเจอร์หรือไลบรารีต่าง ๆ ในโปรเจ็กต์ง่าย ๆ ผ่านการเพิ่ม gem ลงในไฟล์ Gemfile และใช้คำสั่ง bundle install.
