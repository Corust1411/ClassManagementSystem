Angular

React

เป็นเฟรมเวิร์ก JavaScript สำหรับสร้าง UI ที่พัฒนาโดย Facebook. เน้นความสะดวกในการสร้าง component และจัดการสถานะ (state) ของแอป.

Tailwind

เป็นเฟรมเวิร์ก CSS ที่ไม่มีการกำหนดไว้ล่วงหน้า (utility-first). ให้ความสามารถในการสร้างสไตล์ที่ทันสมัยและมีประสิทธิภาพ. ช่วยให้สร้างเว็บไซต์ที่ทำงานได้บนหลายอุปกรณ์.
