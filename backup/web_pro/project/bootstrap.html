Bootstrap

Bootstrap เป็นเฟรมเวิร์ก CSS, HTML, และ JavaScript ที่ใช้สำหรับการพัฒนาเว็บไซต์และเว็บแอปพลิเคชัน. ถูกสร้างขึ้นโดยทีมพัฒนาที่ Twitter และได้รับความนิยมมากในวงการพัฒนาเว็บ. Bootstrap มีคุณสมบัติหลายอย่างที่ทำให้นักพัฒนาสามารถสร้างหน้าเว็บที่สวยงามและปรับแต่งได้ง่ายขึ้น โดยไม่จำเป็นต้องเขียน CSS หรือ JavaScript จากศูนย์. นอกจากนี้ยังช่วยในการทำให้เว็บไซต์เป็น responsive ด้วยการให้ grid system ที่สามารถปรับขนาดได้ตามอุปกรณ์ที่ใช้ 

 Example Code CSS
.container{
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  margin-top: 8px;
  padding-top: 8px;
  z-index: 10;
  width: 100%;
  margin-top: 8px;
  height: 100%;
  border: 1px solid #475569; 
  border-radius: 0.5rem;
  box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06); 
}

@media (min-width: 768px) {
  .container {
    flex-direction: row;
  }
}

Bootstrap เป็นเฟรมเวิร์กที่มีข้อดีมากมายที่ทำให้นักพัฒนาและออกแบบเว็บไซต์มักทำนิยมใช้:


  • Responsiveness (การทำให้เว็บไซต์ทำงานได้บนหลายขนาดหน้าจอ): Bootstrap มี Grid System ที่ช่วยให้เว็บไซต์ทำงานได้ดีในหลายขนาดหน้าจอต่าง ๆ ตั้งแต่แท็บเล็ตไปจนถึงมือถือ

  • สไตล์ที่สวยงาม: Bootstrap มีสไตล์ที่ถูกออกแบบมาเพื่อดูดีและมีลักษณะทันสมัย ทำให้เว็บไซต์ที่สร้างด้วย Bootstrap มีลุคที่ดีตา.

  • Cross-Browser Compatibility (ความเข้ากันได้กับเบราว์เซอร์): Bootstrap ถูกทดสอบเพื่อความเข้ากันได้กับเบราว์เซอร์หลายชนิด, รวมถึง Chrome, Firefox, Safari, และ Internet Explorer.
